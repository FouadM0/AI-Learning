# Automated Financial Data Analytics Pipeline

An end-to-end Python pipeline that engineers synthetic financial data, automates data cleaning, computes key corporate finance metrics (YoY Growth, EBITDA Margins), and generates executive-ready visual trend charts.

## Project Structure
* `day1_automation.py` - Synthesizes raw data and executes programmatic data imputation.
* `day2_analysis.py` - Aggregates monthly data into annual cycles and calculates YoY growth rates.
* `day3_visualization.py` - Generates trend lines and performance comparisons.
* `cleaned_financial_data.csv` - The processed corporate dataset.
* `annual_finance_summary.csv` - High-level executive performance summary.

## Technologies Used
* Python 3.13
* Pandas & NumPy (Data Engineering)
* Matplotlib & Seaborn (Data Visualization)
* Git/GitHub (Version Control)
* GitHub Copilot (AI-Assisted Development)
